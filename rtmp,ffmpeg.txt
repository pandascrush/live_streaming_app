1. this is the command to run a local file :
ffmpeg -re -i C:\path\to\your\file.mp4 -c:v libx264 -preset fast -b:v 1000k -c:a aac -b:a 128k -f flv rtmp://localhost/live/stream


2. video only showing :
ffmpeg -f dshow -i video="Device Name" -c:v libx264 -preset veryfast -f flv rtmp://localhost/live/stream


3. video + audio : 
ffmpeg -f dshow -i video="Device Name":audio="Audio Device Name" -c:v libx264 -preset veryfast -f flv rtmp://localhost/live/stream


4. getting the informationa about which device connected in our system;
ffmpeg -list_devices true -f dshow -i dummy


5. this command is using to trigger the ffplay coming from ffmpeg: converting rtmp to http
